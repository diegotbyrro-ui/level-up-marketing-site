# Site Level Up Marketing Digital

Site institucional estático, responsivo e pronto para GitHub Pages ou hospedagem tradicional.

## Teste local

```powershell
python -m http.server 5500
```

Acesse `http://localhost:5500`.

## Publicação

O pacote principal inclui um script PowerShell que cria o repositório e envia para o GitHub. Para o servidor, envie o conteúdo deste projeto para a pasta `public_html`.
